# 🍕 Pizza Sales Analytics using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on transactional pizza sales data to uncover business insights, customer purchasing patterns, and sales trends. Using Python and data analysis libraries, the project evaluates key performance indicators (KPIs) that help support data-driven business decisions.

---

## Business Problem

Pizza stores generate thousands of transactions every month. Analyzing this data helps answer important business questions such as:

- Which pizzas generate the highest revenue?
- Which pizza categories are most popular?
- What are the busiest sales hours?
- Which months generate maximum revenue?
- What is the average customer order value?

---

## Dataset

The dataset contains transaction-level pizza sales information including:

- Order ID
- Pizza Name
- Pizza Category
- Pizza Size
- Quantity
- Total Price
- Order Date
- Order Time

- Hourly Sales Trend

- Bottom 5 Least-Selling Pizzas

---

## Business Insights

The analysis helps management:

- Identify peak business hours for staffing decisions.
- Understand customer preferences by pizza category and size.
- Recognize top-performing menu items.
- Identify underperforming pizzas for menu optimization.
- Track sales trends over time for better business planning.
